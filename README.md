# Device DB Site

## Как запустить

1. Положите свой файл `2lr.db` в папку `db/`.
2. Установите Flask:
    ```
    python -m venv venv
    source venv/bin/activate
    pip install flask
    ```
3. Запустите приложение:
    ```
    python app.py
    ```
4. Откройте браузер: [http://localhost:5000](http://localhost:5000)
